Cultural Gimbap Website

The Gimbap website responds to the design prompt in Design Studio regarding cultural engagement. The assignment aimed for students to find a new way to recreate their own cultural experiences. The website aimed to inform the cultural item of Gimbap -- a traditional Korean food. By coding HTML, CSS, and JS, I managed to create a Gimbap icon website for personalized Gimbap icons.

Process

I first created a small prototype with some rudimentary code to create the website.

Problems

1. Icons can be branded to a specific aesthetic
2. Have cooking process of Gimbap and description

Next Step

After these two major criticisms, I recreated the website to have a more standard identity, along with pages that include ingredient and cooking process page.

Reflection

Project allowed me to inspect different ways to engage audience using an interactive website. Project also allowed me to create identity of website for an interactive experience.
