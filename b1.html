Busan Sea Festival

Branding Project

Busan Sea Festival is a branding identity project that aims to promote the annual sea festival in Busan, South Korea. Being inspired by the sea waves, I converted one wave into a quarter-ellipse. This quarter ellipses soon compounded to create many different patterns that emulate a geometric tile-like designs. Soon enough, these ellipses combined to create a blue geometric Korean letter forms - forms that are only specific to this poster. Additionally, these ellispes would create fish that would swim within the ocean. All of these combine to create a vertical poster that is both delicate and has a strong geometric identity.
