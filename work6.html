2022 South Korean Election

NYT Interactive Inspired Website

2022 South Korean Election website is a website that I have created for CTC Artist Report. The CTC Artist Report focused on creating interactive experiences in a way that can be presentable to newspapers or magazines. The class invited guest editors and journalists that would help impact and create this website. The website focuses on the demographic breakdown of the 2022 South Korean election.

Process

I first created a page with HTML CSS after I prototyped it with Figma. The demographic was split into three divisions: age, gender, and region. The speech bubbles are the certain language used during the election. The language aspect was meant to differentiate it from the different demographic pages. Specifically in the election 2022, internet language affected the way and swayed the results.

This would be the diagram page of the website that is inspired by the NYT layout. You could see some of the digital coding skecthes I created (that repeated text) to recreate the candidates' faces.

This would be a pop-up page that would create an interactive experience for each division (all having some relation to language)

Problems

After some user testing, I have collected the following criticism

1. Experience can be tied to one page
2. Shorten the experience to its essential

Next Step

With these criticisms in mind, I recreated the page with HTML/CSS/JS to tie the experiences to a single page and a continuous line that ties the narrative together.

Reflection

I learned to create a narrative webpage in an unique interactive manner.
