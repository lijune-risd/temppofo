Tour Website

Subterranean -- a fictional brand I created in my Color/Surface class at Spring 2022 -- became a website project I have decided to progress. The project became a way to explore creating a standard website page (with tour reservation features). Using the brand identity I have created, I extended the aesthetic to this website. In this website, I utilized Slider JS and HTML/CSS.
